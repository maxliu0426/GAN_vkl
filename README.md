# GAN_vkl
implementation of GAN-vkl

## Prerequisites

- Python 3.3+
- Tensorflow
- SciPy
- pillow
- (Optional) Large-scale CelebFaces Dataset



## Usage

First, download the inception model

    $ python download.py --outfile inception_score.model
    
To train a model with downloaded dataset:

    $ python main.py
